Designed and programmed an embedded system using Arduino Uno with LDR light sensor and ultrasonic proximity sensor. Output displayed on 16x2 LCD I2C display with LED indicators. Simulated and tested in Tinkercad.

// Arduino — Proximity Detector (3-pin ultrasonic)
#define SIG_PIN 7
void loop() {
pinMode(SIG_PIN, OUTPUT);
digitalWrite(SIG_PIN, HIGH); delayMicroseconds(10);
digitalWrite(SIG_PIN, LOW);
pinMode(SIG_PIN, INPUT);
long dur = pulseIn(SIG_PIN, HIGH);
float dist = dur * 0.034 / 2;
lcd.print("Dist: "); lcd.print(dist); lcd.print("cm");
}

ACHIEVEMENT_02
SOLIDWORKS — 3D Mechanical Design & Assembly

Used SOLIDWORKS to design 3D mechanical parts, assemblies, and technical engineering drawings. Applied constraints, mates, and simulation tools to validate structural integrity and mechanical fit.

# SOLIDWORKS Design Workflow
1. Sketch 2D profile → Define dimensions & constraints
2. Extrude / Revolve → Create 3D solid body
3. Add fillets, chamfers & holes features
4. Assembly → Apply mates (coincident, parallel)
5. Simulation → Check stress & motion analysis
6. Export → Engineering drawing (DXF / PDF)
